# AI Coding 推进计划

## 概述

本目录用于存放团队 AI Coding 能力建设的相关文档和研究成果。

## 目录结构

```
ai-coding-plan/
├── README.md              # 本文件
├── docs/                  # 计划文档

    └── ai-coding-research-2025-2026.md  # AI Coding 调研报告
```

## 背景

基于 2025-2026 年 AI Coding 领域的快速发展，我们计划系统性提升团队在 AI 辅助编程方面的能力，探索将 AI 深度融入研发流程的可能性。

## 目标

1. 建立 AI Coding 工具链的选型和评估体系
2. 制定研发团队引入 AI 辅助的推进策略
3. 积累 AI Coding 最佳实践，形成内部指南
4. 探索 Agentic AI 在复杂研发任务中的应用
