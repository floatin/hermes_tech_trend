## 🌏 国内

### 1. RLHF 2.0 用多智能体协同让"奖励黑客"无处遁形
📰 腾讯云开发者社区 | 07-01 08:31 | https://cloud.tencent.com/developer/article/2663599
RLHF 2.0 在传统 SFT→Reward Model→PPO 三阶段之上引入多智能体协同机制:多个 Critic Agent 对同一生成结果交叉验证奖励信号,缓解单一 RM 的"奖励黑客"问题。同源工作 DPO 直接跳过 RM、RLAIF 用 AI 反馈替代人工标注。对齐技术从"人类反馈单一通道"向"多通道+自动化"演进,适合企业级 Agent 系统的偏好对齐场景。

### 2. ICLR 2026 Oral:ExDM 探索式扩散模型,无监督强化学习新范式
📰 GitHub / ICLR 2026 Oral | 06-04 | https://github.com/yingchengyang/ExDM
ExDM(Exploratory Diffusion Model)针对无监督强化学习的稀疏奖励难题,提出基于扩散模型的探索策略取代传统离散噪声。在 URLB 基准的 Maze、Quadruped 等任务上取得 SOTA。核心创新是用扩散过程的多样性天然生成机制做探索,与传统 RL 的 ε-greedy / OU noise 相比,在长视野任务上样本效率显著提升。配套一键 conda 复现。

### 3. SSRL 自搜索强化学习:Agentic RL 的"Sim2Real"时刻
📰 量子位 / 上海 AI Lab + 清华 + 上交大 | 09-02 | https://new.qq.com/rain/a/20250902A032DL00
清华周伯文教授团队提出 SSRL(Self-Search RL),用结构化 prompt + format reward 让 Agent 通过自我搜索+验证循环完成无外部标注的策略学习。第一作者为上海 AI Lab 博士生樊钰辰。在 Web Agent、工具调用、代码生成任务上,SSRL 以"自证自洽"模式逼近 PPO/GRPO 性能,摆脱对人类示范和标注数据的依赖,降低 Agent 训练成本。

### 4. ICLR 2026 Oral:Aurora 多模态时序预测,首个通用时序基础模型
📰 CSDN / 华东师大 + ICLR 2026 Oral | 05-21 | https://blog.csdn.net/Aifuyao/article/details/160136898
Aurora 是首个多模态时间序列基础模型,在跨域多模态时序语料上预训练,融合时序、文本、图像三种模态。模态引导多头自注意力把文本/图像中的领域知识显式注入时序特征,原型引导流匹配解码实现零样本生成式概率预测。在单模态、多模态、跨域场景均达 SOTA,适合交通、气象、能源、金融等需融合外部知识的时间序列场景。
